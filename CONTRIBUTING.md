# Contributing to Parmana

Thank you for your interest in contributing! Parmana is a community project — every contribution matters.

## How to Contribute

### Reporting Bugs
1. Check if the issue already exists in [GitHub Issues](https://github.com/EleshVaishnav/PARMANA/issues)
2. If not, open a new issue with:
   - Your OS and RAM
   - Steps to reproduce
   - Expected vs actual behavior
   - Terminal output/error

### Suggesting Features
Open an issue with the label `enhancement` and describe:
- What problem it solves
- How you'd expect it to work

### Submitting Code
1. Fork the repo
2. Create a branch: `git checkout -b feature/your-feature`
3. Make your changes
4. Test it works locally
5. Submit a Pull Request

## Areas We Need Help
- Better hardware detection (VRAM support)
- More language support
- Voice input/output
- Discord integration
- Documentation improvements

## Code Style
- Keep it simple and readable
- Comment complex logic
- Test before submitting

## Questions?
Open an issue or reach out on GitHub Discussions.
